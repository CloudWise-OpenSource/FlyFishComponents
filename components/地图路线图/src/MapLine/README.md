
# 这是什么
    这是以中国地图为板块的不同地点路线图
# 这能做什么
    可以直观看到全国不同需求下需要展示的路线图
# 这适用于什么场景
    1、各个航线路线
    2、各个高铁或火车路线
    3、项目路线，如：国网西电东送路线
    4、行程路线，如：多个疫情感染者的全国行程路线
